# Security policy

If you discover a security issue with the Composer plugin or the
underlying `@composer-app/mcp` server, please report it privately —
do not open a public GitHub issue.

Email: josh@hyperact.ing

We'll acknowledge within 72 hours and work with you on a coordinated
disclosure timeline.

## Scope

- The `composer-mcp` MCP server (`@composer-app/mcp` on npm).
- This plugin's manifest, slash commands, skills, and any bundled
  configuration.
- The Composer service at `usecomposer.app` (in scope, but report
  service-side issues directly via the email above).
